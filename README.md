# Kashiwazaki SEO All Content Lister

![WordPress](https://img.shields.io/badge/WordPress-5.0%2B-blue.svg)
![PHP](https://img.shields.io/badge/PHP-7.4%2B-purple.svg)
![License](https://img.shields.io/badge/license-GPLv2%2B-green.svg)
![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)

全ての投稿・固定ページ・カスタム投稿タイプを一覧表示し、ソート・フィルター機能を提供するWordPress管理ツールです。

## 機能

- 全投稿タイプのコンテンツを一覧表示
- 投稿タイプ・ステータスによるフィルター
- カラムによるソート
- 表示カラムの切り替え
- CSVエクスポート（UTF-8 / Shift-JIS）
- SEOメタ情報の表示（キーワード・ディスクリプション）
  - Yoast SEO
  - All in One SEO
  - Rank Math
  - SEOPress
  - Kashiwazaki SEO Auto Description

## 動作要件

- WordPress 5.0以上
- PHP 7.4以上

## インストール

1. プラグインフォルダを `/wp-content/plugins/` にアップロード
2. WordPress管理画面の「プラグイン」からプラグインを有効化
3. 管理メニューに「Kashiwazaki SEO All Content Lister」が追加される

## 使い方

1. WordPress管理画面のサイドメニューから「Kashiwazaki SEO All Content Lister」をクリック
2. 投稿タイプやステータスでフィルターして表示
3. 必要に応じてCSVエクスポート
