== Description ==

Kashiwazaki SEO All Content Listerは、WordPressサイト内の全てのコンテンツを一覧表示できる管理ツールです。

= 主な機能 =

* 全投稿タイプのコンテンツを一覧表示
* 投稿タイプ・ステータスによるフィルター
* カラムによるソート
* 表示カラムの切り替え
* CSVエクスポート（UTF-8 / Shift-JIS）
* SEOメタ情報の表示（キーワード・ディスクリプション）

= 対応SEOプラグイン =

* Yoast SEO
* All in One SEO
* Rank Math
* SEOPress
* Kashiwazaki SEO Auto Description

== Installation ==

1. プラグインフォルダを `/wp-content/plugins/` にアップロード
2. WordPress管理画面の「プラグイン」からプラグインを有効化
3. 管理メニューに「Kashiwazaki SEO All Content Lister」が追加される

== Frequently Asked Questions ==

= どの投稿タイプが表示されますか？ =

公開設定されている全ての投稿タイプ（投稿、固定ページ、カスタム投稿タイプ）が表示されます。

= CSVの文字コードは選べますか？ =

UTF-8とShift-JISから選択できます。Excelで開く場合はShift-JISを推奨します。
